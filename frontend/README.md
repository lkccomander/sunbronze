# SunBronze Frontend

Next.js + TypeScript + Tailwind frontend for the receptionist/admin interface.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Copy environment values:

```bash
cp .env.example .env.local
```

3. Start the app:

```bash
npm run dev
```

## API target

Set the backend base URL in `.env.local`:

```env
NEXT_PUBLIC_API_BASE_URL=https://your-api-domain
```
